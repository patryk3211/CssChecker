Sprawdzacz CSS

Opis

Program ten działa szukając różnic w stylach przypisanych do różnych elementów. Aby porównywanie działało niezależnie od różnic w nazwach oraz sposobach nadawania styli Należy dokonać kilku założeń:

  1. Id lub klasy przypisane są do div-ów
  2. Klasy w dokumencie mogą być użyte tylko raz
  3. Nie liczy się kolejność elementów w dokumencie

Działanie

Aby móc w łatwy sposób wyszukiwać różnice w stylach elementów pliki CSS są przetwarzane w graf.
